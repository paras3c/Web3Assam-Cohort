## **4. API Endpoints**

### **POST /api/v1/user/register**  
Creates `data.json` and saves user-entered data.

### **GET /api/v1/user/get**  
Returns stored user data.

### **PUT /api/v1/user/update**  
Updates and overrides existing user data in `data.json`.
