/mentee/SUMIT DAS/Assignment-1/

  frontend/    # React (Vite) frontend  (Local:   http://localhost:5173/ )

## How to run

### Backend
cd backend
npm install
npm start
# Server runs on http://localhost:5000

### Frontend
cd frontend
npm install
npm run dev
# Open the dev URL shown (default http://localhost:5173)

## Technologies Used
- Node.js + Express
- React + Vite
- FS module (fs.promises)
- CORS

## API Endpoints
- GET /api/user - Returns the current user JSON.
- PUT /api/user - Update user fields: name, email, role.
